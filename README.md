# GPS Python
### By Florian Reimat
## About 
This repo is an .nmea to .kml converter, it can work with:
- [x] GGA 
- [x] VTC
- [x] RMC
- [ ] ZDA
- [ ] GSV 
- [ ] GLL 

This were a classes work in DUT GEII of Toulon (2019)
